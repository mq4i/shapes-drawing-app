## Python Project: Shapes Drawing App

Description
- A graphical application built with Python Turtle module that draws multiple geometric shapes using object-oriented programming. Each shape is implemented as its own class with customizable size, color, and position. The app also draws a small house with a roof, door, and door handle.

Shapes Drawn
- Square
- Circle
- Triangle
- Rectangle
- Hexagon
- Pentagon
- Star
- Decagon
- House

How to Run
1. Run the Python file
2. A Turtle graphics window will open automatically.
3. All shapes will be drawn on screen with their details printed in the terminal.

Features
- Object-oriented programming
- Customizable fill color, size, and position for every shape
- Prints shape details (size, color, position) to the terminal
- Draws a house shape with body, roof, door, and door handle
- Clean and organized code structure

Requirements
- Python 3 installed
